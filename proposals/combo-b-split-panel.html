Why this combination works
Arch fix The right panel is always present — Create, Stash, and Stats are never separate landing pages. Tool pages only open with an active project.
Projects first The entire left column is permanently dedicated to projects. Nothing competes with it for space — no greeting section, no QuickTiles, no StashSummary card.
Always visible Unlike tabs, the action panel is always visible — no need to "find" where to create a new project or check the stash. Reduces navigation decisions.
No duplication Every action appears exactly once. Active project in the list + accent bar replaces GreetingHero + ResumeCard + RecentGrid as separate sections.
