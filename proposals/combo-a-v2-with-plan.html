Restructure the Projects tab in home-app.js
home-app.js
Three changes in one pass:

3
Build the Create / Stash / Stats tab panels in home-app.js
home-app.js
These replace the separate-page landing states:
  • Create new tab — two tiles: "New from image" (index.html?action=new-from-image) and "New from scratch" (index.html?action=new-blank). These still navigate to the Creator — the difference is the user explicitly chose to create, rather than being landed there.

  • Stats tab — a single link/button to index.html?mode=stats. Can be expanded later.
