What this option eliminates
✕
Top tab bar that duplicated page navigation — the sidebar lists all tools and projects directly, with no duplication. Each destination appears exactly once.
✕
GreetingHero + ResumeCard as separate sections — State A shows a single large resume hero. The greeting is the hero's headline, not a standalone section above it.
✕
QuickTiles "Open Tracker" + "Open Stash Manager" — these are sidebar links. They no longer appear as tiles on the home page.
✕
Confusing redirect-to-home guard behaviour — the guard existed because tool pages (Tracker, Creator) were doubling as landing pages. With a context-aware home, tool pages only open with a project active — the home page IS the landing page, always.
✕
StashSummary on home page — stash stats live in the sidebar / Stash Manager page. The home page focuses on project work.
✕
home-screen.js legacy component — the new State B welcome screen replaces the old home-screen.js "empty home" without any of the duplication.
