Proposal D — Rich Hover Cards with Pin
  • Hover the info icon next to any setting for a rich card: title, explanation, live example data, and a "Read more" link.
  • Click the pin icon to keep a card open while you work with both hands.
  • Pinned cards gain a close button. Multiple cards can be pinned simultaneously.
  • Full documentation lives at a separate deep-linkable help URL; "Read more" opens the relevant article.

Removes isolated single-colour stitches ("confetti") that are surrounded by stitches of different colours. Each confetti stitch requires a thread change — removing them speeds up stitching significantly.

Dithering mode
Controls how colours between your palette entries are approximated. Floyd-Steinberg diffuses the colour error to neighbouring cells, producing smooth-looking gradients. Leave at the default unless you prefer a more graphic, flat look.
