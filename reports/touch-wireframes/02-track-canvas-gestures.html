02 — Track-mode canvas, iPad landscape (1024×768)

Track mode • iPad landscape • Lpanel in RAIL state • bottom action bar with Mark/Hand/Range

Key changes: the lpanel collapses to a 56 px rail showing the active highlight swatch and a legend jump (resolves "left panel takes too much room"). The bottom action bar gains explicit Mark / Hand / Range tools — the user is never forced to remember what 1-finger drag does because the active sub-tool is highlighted. 2-finger pan and pinch always work, regardless of tool. Long-press opens a context menu (info, focus block, "go to next of colour") instead of committing a destructive range (resolves T-2).
