01 — Edit-mode canvas, iPad landscape (1024×768)

Edit mode • iPad landscape • Hand tool added • toolbar at bottom

What's new vs. today: the toolbar moves to the bottom (thumb-zone), hits the 44 px target floor with 8 px gaps, and gains an explicit Hand tool plus a Focus entry. With the Hand tool active, 1-finger drag pans (resolves E-1). With Paint active, 1-finger drag brush-strokes with a 10 px slop; 2-finger drag still pans regardless of tool. Long-press now opens the cell context menu in any tool state (resolves E-3).
