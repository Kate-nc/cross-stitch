06 — One-time gesture guide overlay (first touch session)

Shown on first touch session per device; persists in UserPrefs gestureGuideShown

One-time gesture guide. Appears once per device class on first touch input on the canvas (detected via the first non-mouse pointerdown). The "Got it" button dismisses and writes gestureGuideShown=true; users can re-open it from Help. Keeps the universal-rules table memorable without forcing every user to read documentation.
