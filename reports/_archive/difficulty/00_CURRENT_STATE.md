## 2. Exact Implementation (verbatim)

```javascript
function calcDifficulty(palLen,blendCount,totalSt){
  let score=0;
  if(palLen<=8)score+=1;else if(palLen<=15)score+=2;else if(palLen<=25)score+=3;else score+=4;
  if(blendCount>0)score+=1;if(blendCount>5)score+=1;
  if(totalSt>10000)score+=1;if(totalSt>30000)score+=1;
  if(score<=2)return{label:"Beginner",color:"var(--success)",stars:1};
  if(score<=4)return{label:"Intermediate",color:"#A06F2D",stars:2};
  if(score<=6)return{label:"Advanced",color:"var(--accent-hover)",stars:3};
  return{label:"Expert",color:"var(--danger)",stars:4};
}
```

## 10. What the Current Model DOES NOT Capture

1. **Confetti density** — a 40-colour pattern with 40-stitch blocks is profoundly easier than one where every stitch changes colour. Completely absent from the model.
2. **Colour changes per row / per unit area** — no measurement of how frequently the stitcher must switch threads.
3. **Average run length** — how many contiguous same-colour stitches exist before a colour change.
4. **Perceptual colour similarity** — two nearly-identical greens are harder to work with than two very different colours; not considered.
5. **Fabric count / physical size** — 28-count requires better eyesight and more care; 14-count is forgiving. Ignored.
6. **Backstitch complexity** — `bsLines` data exists but is not used.
7. **Fractional / specialty stitches** — `halfStitches` data exists but is not used.
8. **Colour distribution** — a pattern with 3 dominant colours (90% of stitches) and 37 accents is easier than 40 evenly-distributed colours.
9. **Physical dimensions** — a 200×200 pattern on 14-count is 36 cm² of fabric to manage; a 50×50 is portable. Size affects handling, not just time.
10. **Estimated hours** — no time estimate surface (users must guess).
11. **Stitcher-relative adjustment** — no personalisation based on stitcher experience, prior projects, or preferred fabric count.

## 12. Summary for Agent Teams

The current calculator is a 3-parameter, 8-point additive model returning a 4-tier label. It is simple, fast, and partially correct — palette size and stitch count do correlate with difficulty — but it misses the most impactful real-world factors: confetti density, thread-change frequency, perceptual colour similarity, and fabric count. All downstream consumers expect a `{label, color, stars}` object; any new model must preserve or upgrade this contract.

The full stitch map (`pattern` array, length `w*h`) is available at calculation time, enabling spatial analysis such as run-length measurement and colour-adjacency computation. The DMC Lab values in the `DMC` global enable perceptual distance calculations between palette colours.
