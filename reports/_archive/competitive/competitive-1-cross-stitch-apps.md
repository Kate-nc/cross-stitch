# Competitive Report 1: Cross-Stitch App Landscape

> **Research basis:** LordLibidan.com (updated March 2026, ~20 000 verified reviews);
> StitchMate comparison pages; direct tool inspection.
> Scores are LordLibidan's aggregated user ratings unless noted.

---

## 1. Market Segments

The cross-stitch software market splits cleanly into four delivery models, each
with different user expectations and pricing norms:

| Segment | Pricing norm | User expectation |
|---|---|---|
| Desktop installers | One-time $30–$155 | Full feature set, offline, printable PDF |
| Web tools | Free–$10/pattern or $5–$15/month | Zero install, immediate results |
| Mobile tracking | Free–$18/year | Quick mark-up on-device while stitching |
| Mobile creation | $8–$15 one-time | Lightweight pattern making on the go |

No single product credibly straddles all four. The primary user journey is:
**create on desktop or web → track on mobile**. Serving that full arc in one
free, offline-capable web app is the position our app uniquely occupies.

## 2. Desktop Creation Tools

### WinStitch / MacStitch — 10/10 (LordLibidan)
- **Price:** $52 (~$40 with affiliate discount)
- **Platforms:** Windows & macOS (separate binaries, cross-project compat)
- **Threads:** 30 brands
- **PDF export:** Yes — full feature set
- **Updates:** Yearly (paid upgrade, not required)
- **Standout:** Opens most PCStitch files; Hardanger stitch library; pattern
  cross-device transfer between Win and Mac builds
- **Weakness:** $40–$52 barrier; no web/mobile companion; desktop-only

### PCStitch — 8.5/10
- **Price:** $50 (~$20 with code)
- **Threads:** DMC, Anchor, others
- **PDF export:** Yes
- **Updates:** Bug fixes only since 2016; no new features
- **Standout:** 2 000 pre-programmed pattern library; strong colour-picking tools
- **Weakness:** Proprietary `.pcs` format, can only be opened by PCStitch
  viewer; stagnant feature set; no cross-platform support

### KG Chart — 7/10
- **Price:** $35
- **Threads:** DMC + Cosmo
- **PDF export:** No native export (requires free third-party CutePDF)
- **Updates:** Quarterly
- **Standout:** .oxs format widely supported; large community
- **Weakness:** Colour accuracy can skew red; growing number of newer apps
  that cannot open KG Chart files; PDF gap is a genuine friction point

### MiniStitch — 7/10 (free)
- **Price:** Free (from WinStitch makers)
- **Threads:** DMC only
- **PDF export:** Yes
- **Updates:** Yearly
- **Standout:** "First step on the journey" — robust core behind simple UI
- **Weakness:** Limited compared to paid WinStitch sibling

### Ortmann Cross Stitch Designer — 7/10
- **Price:** $39–$189 (tiered)
- **Threads:** DMC (447 colours), Anchor, Madeira, Blomstergarn
- **PDF export:** Yes
- **Updates:** Yearly
- **Weakness:** Key features (cover page, designer branding, title pages) only
  available on $69–$189 tiers — 5× price jump for parity features

### ILSoft Stitch R-XP — 7/10
- **Price:** $103
- **Threads:** DMC + Anchor
- **Updates:** Last updated 2009; still used by DMC France
- **Weakness:** Significant stagnation; high price for no active development

### Others (low-rated or abandoned)
- **Istitch 2** ($30) — no updates, Jane Greenoff's product
- **DP Software Cross Stitch Pro Platinum** ($89–$130) — 20 brands, yearly
  paid upgrade, "VERY expensive considering alternatives"
- **StitchCraft** ($155–$2595) — 33 brands, never updated, cited as overpriced
- **Stitch Painter Gold** ($199) — requires physical USB dongle at all times
- **HaftiX, Ryijy, Cstitch, pixelcross** — free, open-source or unmaintained

**Desktop segment summary:** WinStitch/MacStitch is the clear leader for quality.
PCStitch has a loyal user base but is coasting on inertia. The rest are either
declining or niche. The segment has low innovation velocity overall.

## 3. Web / Online Creation Tools

### StitchMate — 9.5/10 (LordLibidan, April 2026: "best online program right now")
- **Price:** Free to create; £1.60/pattern (10-pack), £3.49 single, or passes
  from £47.99/year
- **Threads:** 50+ brands (DMC, Anchor, Cosmo, Madeira, hand-dyed, etc.)
- **PDF export:** Pattern Keeper–certified; vector-sharp symbols
- **Updates:** Active development
- **Standout features:**
  - **FLOW Score (0–100):** Spatial-aware stitchability quality metric shown in
    real time during generation
  - **ConfettiScope:** Visual overlay highlighting isolated stitches; 79% fewer
    confetti than Stitch Fiddle (5.1% vs 15.8%) on identical inputs
  - **Blended threads** (2–4 strands, across all brands)
  - **French knots**
  - **Layers** for non-destructive editing
  - **Import:** .oxs, .pat, .xsd (broader than any other web tool)
  - **Text tool:** 48 stitch-friendly fonts, real-time preview
  - **Commercial licensing:** brandless exports, custom PDF branding for Etsy
    sellers
  - Full editor free (pay only to export PDF)
- **Weakness:** No stash tracking; no progress tracking; single-craft (cross
  stitch only); pay-per-export model adds up for prolific users

### Thread-Bare — 10/10 (LordLibidan, previously top-ranked)
- **Price:** $10 per pattern
- **Threads:** DMC, Anchor, Riolis
- **PDF export:** Yes
- **Updates:** Monthly
- **Standout:** "First-class" dithering, described as superior to any paid
  desktop tool; well-built interface; strong toolset
- **Weakness:** Per-pattern pricing punishes active users ($120+/year for one
  pattern per month); daunting for beginners; limited brand selection

### Stitch Fiddle — 10/10 (free)
- **Price:** Free; Premium ~$5.50/month
- **Threads:** DMC + Anchor (~30 brands)
- **PDF export:** Yes (free); Pattern Keeper compat: No
- **Standout:**
  - 2 000 × 2 000 stitch grid (largest publicly stated limit of any web tool)
  - **Multi-craft:** also handles knitting and crochet patterns
  - Built-in **progress tracking** with row markers
  - Very beginner-friendly
- **Weakness:** 15.8% confetti rate on portraits vs Stitchmate's 5.1%; no
  French knots; no blended threads; no layers; backstitch requires Premium;
  OXS import only

### Love It Stitch It — 8/10
- **Price:** $10/month
- **Threads:** DMC only
- **Target:** Small pattern store operators, not casual makers
- **Weakness:** Monthly recurring fee; limited thread choice

### patterncreator.com — 7.5/10
- **Price:** $7/pattern
- **Threads:** DMC, Anchor, Venus
- **Hard limit:** 250 × 250 stitches — "MAJOR issue" at this price point

### patternsforyou.com — 6/10
- **Price:** Free
- **Limit:** 300 × 300 stitches

### pixel-stitch.net — 5.5/10
- **Price:** Free
- **No size limit** but no image editing tools

### FlossCross — 5.5/10
- **Price:** Free; 300 × 300 limit; strong for pixel art

### stitchboard.com / FreePatternWizard — 5/10
- **Price:** Free; 4 brands (DMC, Anchor, Madeira, Red Heart); no image editing

### DMC Snap & Stitch — 5/10
- Buys threads, gets free pattern; limited UX

### knytstudio — 4.5/10
- AI "create from description" feature noted; limited pattern quality

### craftdesignonline — 4.5/10
- 100 × 100 limit; youth-oriented; sharing feature

### Crosti — 2/10
- Wide brand support (8 brands including Gamma, Silk Mori) but poor quality output

**Web segment summary:** StitchMate has overtaken Thread-Bare as the quality
leader for creation. Stitch Fiddle remains the volume leader (price:
unbeatable; grid: very large). The key battleground is **confetti reduction**
and **Pattern Keeper compatibility** — both are now table-stakes for serious
pattern makers.

## 4. Mobile Tracking Apps

### Pattern Keeper — 10/10 (LordLibidan), 100 000+ Play Store reviews
- **Platform:** Android (primary); iOS version discontinued
- **Price:** $10.50 (free trial, 2 weeks)
- **What it does:**
  - Works with **any PDF** — does not require proprietary format
  - Tracks stitched and unstitched threads per page
  - Parking-method tracking (the **only app** to do so per LordLibidan)
  - Combines multiple pattern pages into one continuous chart
  - Stitch count per colour
- **Why it matters:** This is the de-facto standard for tracking in the
  Android community. "Must have for many cross stitchers across the world."
  The entire industry formats PDF exports to be Pattern Keeper–certified.

### Cross Stitch Paradise — 9.5/10 (free, Android)
- Markup app for XSD files
- Tracks full, half, quarter stitches and backstitch
- Session tracking features
- 6 500+ Play Store reviews

### Cross Stitch Thread Organizer — 9.5/10 ($1, Android)
- Thread inventory with to-buy lists and low-stock alerts
- Developer active on Reddit daily
- 248 Play Store reviews

### Markup R-XP — 9/10 ($18/year, iOS + Android)
- **Unique feature:** Photograph a physical printed pattern → converts to
  mark-up-able digital chart (no PDF required)
- Now subscription-based (was one-time); price has increased rapidly
- 510+ Play Store reviews / 1 900+ iOS reviews

### X-Stitch — 9.5/10 ($3, iOS)
- Tracks threads, aida, needles, and charts together
- Reads charts to produce per-project thread lists
- "Need to buy" list generation

### XStitch Plus — 9/10 ($10, iOS)
- Full project journal with 5 images per project
- **Social media publishing** directly from the app
- 31 iOS reviews (small but passionate user base)

### Thread Stash — 8.5/10 (free, Android — download direct, off Play Store)
- DMC, J&P Coats, Anchor, Sullivans, Weeks Dye Works
- Shopping list; account sync for phone changes
- 184 reviews; active development team

### Cross Stitch World — 9.5/10 (free, iOS + Android)
- Actually a **game** (paint-by-numbers with cross-stitch aesthetics)
- 63 000+ Android reviews; accessibility use case for arthritis sufferers

### Thready — 8/10 ($3, iOS)
- Simple thread tracking with clean interface

### Threadster — 8/10 ($3/month, iOS + Android)
- New; praised for design; subscription cost is the main objection

**Mobile tracking summary:** Pattern Keeper is the unchallenged benchmark.
No single app combines creation + tracking + inventory on mobile. Users
maintain 2–3 separate apps to cover the full workflow.

## 8. Key Takeaways for Subsequent Reports

1. **Confetti / orphan stitches** are the #1 technical pain point in web
   creation tools. StitchMate built its entire marketing around solving this.
   Our app has orphan removal — but no quality metric or visual diagnostic tool.

2. **Pattern Keeper compatibility** is table-stakes for PDF export. We already
   have it.

3. **Stitch Fiddle's real moat** is not its grid size — it is multi-craft
   (knitting + crochet). We cannot compete for that audience and should not try.

4. **No competitor integrates** creation + tracking + stash. This is the
   market's most significant structural gap.

5. **Thread brand breadth** is an unresolved weakness in our creator pipeline
   (DMC-only in the pattern engine due to composite-key schema issue) even though
   the stash manager supports DMC + Anchor.

6. **Social / sharing features** (XStitch Plus, Cross Stitch World) are a
   nascent but growing area. None of our direct competitors do this well.
