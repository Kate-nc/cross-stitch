Phone, tablet, desktop · light + dark

Splash, install prompt, welcome wizard, home dashboard, printed PDF, marketing card, app icons, empty states

Everything outside the three workspaces — the screens that decide whether a user installs at all, plus the printed artefact the app ships into the world.

How to read these

A few notes before you click through.

These are still images.

No JavaScript, no interactions, no real data. Every "screenshot" is rendered HTML/CSS — high enough fidelity to judge the design language but not a working prototype.

Workshop is one direction of three.

Studio (cool, indigo) and Folk (sage + pink) remain on offer in ux-11. The Workshop showcase exists because you said you liked it; it's the most thorough render so you can see what "yes" would actually look like.

Direction is separable from plan.

A visual direction styles the app; a plan reorganises it. Workshop will look the same whether we ship Plan A, B, C or a hybrid. See the parent message for hybrid recipes.

No emojis. British English.

Per AGENTS.md, all icons are inline SVG. All copy uses British spellings (colour, organiser, centre).
